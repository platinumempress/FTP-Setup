# FTP-Setup

A one-page website explaining FTP (File Transfer Protocol).

## Features
- Full-screen background image
- Centered text overlay
- Responsive design
- Clean, modern styling

## Files
- `index.html` - Main website
- `styles.css` - Styling
- `peace.jpg` - Background image

## How to Use
Open `index.html` in your web browser to view the website.
